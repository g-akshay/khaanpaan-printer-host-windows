# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating in this project — opening issues, submitting pull requests,
or engaging in any community space tied to it — you agree to abide by its
terms.

## Reporting

Report any concerns to **support@khanpan.app**. All reports are reviewed
promptly and handled confidentially by the project maintainers. We will
acknowledge receipt within 72 hours.

## Enforcement

Maintainers are responsible for clarifying and enforcing the standards of
acceptable behavior described in the Covenant, and will take appropriate
and fair corrective action in response to any behavior they deem
inappropriate.
